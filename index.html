GitHub

Feel free to navigate to my GitHub page to see my up to date projects.

Java Compiler Clone

A large-scale project consisting of developing a lexical analyzer, parser, abstract syntax tree, symbol table, activation code, intermediate code generation, and finally generating assembly code. All of this is implemented to create a working compiler for a clone of Java called “MiniJava.”

Web Application with database

A team focused project, where I was responsible for developing a functioning website that takes user input and stores the user into a database. With the power of HTML, CSS, JavaScript, Node.js, and MongoDB, I was able to stand up a website that takes user input and stores qualified users. Using concepts such as blackbox testing and boundary testing, I was able to test the effectiveness of what I built. CRUD principles were also applied for the database portion.

Internet of Things Research

Under Dr. Richard G. Epstein, I did research into the security issues regarding the Internet of Things. Topics include vulnerabilities within smart devices, the power grid, and the weaponization of devices connected to the internet. The weaponization includes possible dangers within the advent of "smart cities."
